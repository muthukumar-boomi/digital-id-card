# Digital ID Card UI

A clean, modern digital ID card design built with **HTML5**, **CSS3**, and **Font Awesome**. This component is perfect for profile pages, user directories, or employee badges.

## 🚀 Features
- Responsive card layout
- Gradient background with shadow effect
- Custom fonts and opacity styling
- Font Awesome icons for user and phone
- Profile photo and header image

## 🛠️ Technologies
- HTML5
- CSS3 (Flexbox, custom properties)
- Font Awesome 4.7
